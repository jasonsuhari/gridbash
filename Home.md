# GridBash Wiki

GridBash is a Windows-native terminal grid for running many CLI agents at once.

Start here:

- [README](https://github.com/jasonsuhari/gridbash#readme) for install, usage, controls, profiles, and examples.
- [Contributing](https://github.com/jasonsuhari/gridbash/blob/main/CONTRIBUTING.md) for setup, validation, pull requests, and DCO sign-offs.
- [Roadmap](https://github.com/jasonsuhari/gridbash/blob/main/docs/ROADMAP.md) for the product direction.
- [Issues](https://github.com/jasonsuhari/gridbash/issues) for actionable bugs, tasks, and feature requests.
- [Discussions](https://github.com/jasonsuhari/gridbash/discussions) for ideas, questions, roadmap notes, and design conversation.

Current release roadmap:

- v0.2 Nostromo: public launch polish.
- v0.3 Nebuchadnezzar: agent orchestration workflows.
- v0.4 Holodeck: TUI/runtime experience.
- v1.0 Zion: stable Windows release.
- v2.0 Morpheus: daemon and reattach architecture.
