01

Install globally

Grab the tiny npm launcher and the native package for your platform.

npm install -g gridbash
02

Pick your grid

Use the animated setup screen or launch straight into a layout.

gridbash 2x3 --profile codex
03

Run in parallel

Add isolated worktrees when every agent needs its own checkout.

gridbash 2x3 --profile codex --worktrees
