# Support

Use the README first for installation, usage, controls, profiles, and configuration.

For help:

- Open a bug report for reproducible broken behavior.
- Open a feature request for a specific new capability or workflow.
- Open a question issue when the README does not answer a usage question.
- Follow `SECURITY.md` for vulnerabilities.

When asking for help, include your Windows version, host terminal, GridBash version or commit, launch command, selected profile, and the behavior you expected.

GridBash does not currently provide paid support or guaranteed response times.
